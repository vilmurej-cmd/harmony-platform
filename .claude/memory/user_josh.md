Josh Vilmure is the founder of Vilmure Ventures. He is NOT a coder — he copy-pastes terminal commands only. His philosophy: "Don't hold back. Go above and beyond. Shoot for the stars and past them."

HARMONY is his flagship product — an Emotional Music Composer where users describe a moment, select emotions and instruments, and hear AI-composed music with visual feedback (piano keys, particles, waveforms).

Tech stack: Next.js 16 + TypeScript + Tailwind CSS v4 + OpenAI GPT-4o + Tone.js + Framer Motion
Deployed on Vercel at harmony-platform-seven.vercel.app

He expects ambitious, creative, "above and beyond" work. He values jaw-dropping visual experiences over minimal implementations.
